Remit To Sales Channel Upload Tool:

 

Steps in Uploading Remit To Sales Channel:

 

  1. Prepare an input file, which is of “CSV” format (Comma Separated Format). This format can be get –

·        Open an Excel Spread sheet

·        Add all the data in the format mention in the instructions-

·        First Column – Remit To Sales Channel  

·        Second Column - Account Site Orig System Reference  

 

Note that if any of the above columns are null, please leave the value as null.BUT, if you leave the last column also null, then Micro Soft Excel does not put the commas as desired and the file format may be wrong and you may get errors in the file upload. After the data is added in Excel spread sheet, save it as “Comma Separated Values” format, as shown in the screen shot.
