E-Bill Contacts Upload Tool


This is a self service application, where user can upload E-Billing Contacts associated to the customer for a Bill-to site and corresponding Customer Document ID.

Steps in Uploading E-Bill Contacts:


1. Prepare an input file, which is of CSV format (Comma Separated Format). This format can be obtained like this-
   a).Open an Excel Spread sheet
   b).Add all the data in the format mention in the instructions-
   ----First Column - Contact Last Name
   ----Second Column - Contact First Name
   ----Third Column - Contact Email Address
   ----Fourth Column - Phone Area Code
   ----Fifth Column - Phone Number
   ----Sixth Column - Legacy Account Number (8 Digit AOPS Customer ID)
   ----Seventh Column - Adress Sequence
   ----Eighth Column - Contact ID
   ----Ninth Column - Status Flag
   ----Tenth Column - Customer Document ID
Note that if any of the above columns are null, please leave the value as null. BUT, if you leave the last column also null, then Microsoft Excel does not put the commas as desired and the file format may be wrong and you may get errors in the file upload. After the data is added in Excel spread sheet, save it as "Comma Separated Values" format.

So, a typical upload file, which is in CSV format, when opened in a notepad, looks like this- 


Last Name,First Name,Email Address,Area Code,Telephone,Legacy Account Number,Addr Seq,Contact ID,Flag,Cust Doc ID
last,first,first.last@email.com,561,1119999,46292654,1,,A,11405550
2. Select the file from the local file system, using "Browse" button in the page.

3. Click on "Upload E-Bill Contacts" Button in the page.

4. That's it!

This file will be uploaded into server. Background process will be spawned to read the upload file and insert/update the contact information provided. You can go back to E-Bill Main page to verify the contacts attached to the customer document.
